# Offer Copilot Functional Requirements

## Product Overview

Offer Copilot is an AI-powered career assistant that helps users manage their entire job search journey.

The system supports users from resume preparation, job matching, interview preparation, application tracking, to offer decision-making.

Core workflow:

Resume Upload
→ Job Matching
→ Interview Preparation
→ Application Tracking
→ Offer Decision Support


---

# 1. Resume Management & Analysis

## Description

The system allows users to upload resumes and automatically analyze resume content.

Supported formats:
- PDF
- JPG
- PNG

The system extracts resume information and prepares it for AI analysis.

## Features

- Upload resume files
- Parse resume content
- Extract education and work experience
- Identify skills and achievements
- Generate resume improvement suggestions


---

# 2. Job Description Analysis

## Description

The system analyzes job descriptions and extracts key requirements.

## Features

- Extract job responsibilities
- Identify required skills
- Identify preferred qualifications
- Summarize company and role expectations
- Generate job requirement keywords


---

# 3. Resume-JD Matching Analysis

## Description

The system compares user resumes with target job requirements and provides personalized recommendations.

## Features

- Resume-job matching score
- Identify matching strengths
- Detect missing skills
- Provide improvement suggestions
- Optimize resume keywords for ATS systems


---

# 4. Interview Preparation

## Description

The system helps users prepare for interviews based on target positions and personal experience.

## Features

- Generate potential interview questions
- Generate personalized answer frameworks
- Support STAR method answer preparation
- Simulate interview conversations
- Evaluate answer quality


---

# 5. Job Application Management

## Description

The system helps users manage their entire job search pipeline.

Users can track each job opportunity from application to final result.

## Features

- Save job opportunities
- Record company and position information
- Track application status
- Manage interview progress
- Record important dates
- Set follow-up reminders


## Application Status

Supported stages:

- Applied
- Resume Screening
- Interview Round 1
- Interview Round 2
- Final Interview
- Offer Received
- Rejected


---

# 6. Interview Calendar & Timeline

## Description

The system provides a timeline and calendar view to help users manage interviews and important job-search events.

## Features

- Create interview events
- View upcoming interviews
- Record interview notes
- Track interview history
- Display job-search timeline


Example:

Company A - AI Product Manager

Timeline:

Applied
↓

---

# 7. Offer Management

## Description

The system assists users in evaluating and managing job offers.

## Features

- Store offer information
- Compare multiple offers
- Analyze compensation packages
- Evaluate career development opportunities
- Provide decision support
