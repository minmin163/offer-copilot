# Offer Copilot MVP Design


## 1. Product Goal

Offer Copilot is an AI-powered career assistant that helps job seekers improve their application success rate.

The MVP focuses on resume and job matching analysis.


## 2. Core User Flow


User uploads resume

↓

User uploads target job description

↓

AI analyzes resume and job requirements

↓

System generates matching results and optimization suggestions



## 3. MVP Features


## 3.1 Resume Upload

Description:

Users can upload their resumes for AI analysis.


Supported formats:

- PDF
- JPG
- PNG


Processing:

Resume File

↓

File Storage

↓

PDF Parser / OCR

↓

Text Extraction

↓

AI Analysis



## 3.2 Job Description Upload

Description:

Users can upload target job descriptions.


Supported formats:

- PDF
- JPG
- PNG


Processing:

Job Description File

↓

File Storage

↓

PDF Parser / OCR

↓

Requirement Extraction

↓

AI Analysis



## 3.3 Resume-JD Matching Analysis

Description:

The system compares resume information with job requirements.


Output:

- Match score
- Strengths
- Missing skills
- Optimization suggestions



## 3.4 Interview Preparation

Future feature:

Generate interview questions based on:

- Resume
- Job description
- Company requirements



## 4. MVP Success Criteria


Users can:

- Upload resume files
- Upload job description files
- Receive AI-generated matching analysis


## 5. Future Expansion


Potential features:

- Application tracking
- Interview calendar
- Offer comparison
- Career roadmap
- AI career agent
