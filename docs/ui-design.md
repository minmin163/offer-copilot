# Offer Copilot UI Design


## 1. MVP Main Page

### Goal

Provide users with a simple workflow to analyze resume and job compatibility.

The page allows users to upload resume files and job descriptions, then receive AI-powered career insights.



## 2. User Flow


User enters Offer Copilot

↓

Upload resume

↓

Upload job description

↓

Click Analyze button

↓

Receive AI analysis results



## 3. Page Structure


### Header

Components:

- Product name
- Product description


Content:

Offer Copilot

AI Career Assistant



### Resume Upload Section

Purpose:

Allow users to upload their resume files.


Components:

- Upload button
- File preview
- Upload status


Supported formats:

- PDF
- JPG
- PNG



### Job Description Upload Section

Purpose:

Allow users to upload target job descriptions.


Components:

- Upload button
- File preview
- Upload status


Supported formats:

- PDF
- JPG
- PNG



### Analysis Section

Purpose:

Start AI matching analysis.


Components:

- Analyze button
- Loading status
- Result display
